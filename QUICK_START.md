# Quick Start Guide

## 🚀 View Documentation (3 steps)

```bash
# 1. Go to project directory
cd /workspaces/Mastering-Agentic-AI-with-Java

# 2. Start the server
mkdocs serve

# 3. Open browser
# http://127.0.0.1:8000
```

## 📖 What You'll See

### Left Sidebar - Tree Navigation
- Click sections to expand/collapse
- Navigate through all 5 blocks
- See current page highlighted

### Center - Main Content
- Read documentation
- View images and diagrams
- Copy code examples

### Right Sidebar - Table of Contents
- Jump to sections within page
- Auto-generated from headings
- Follows your scroll

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Focus search |
| `Ctrl/Cmd + /` | Show shortcuts |
| `Arrow Keys` | Navigate pages |

## 🎨 Features

- ✅ **Search** - Type to search all content
- ✅ **Dark Mode** - Toggle with icon (top right)
- ✅ **Copy Code** - Click copy button on code blocks
- ✅ **Zoom Images** - Click any image to zoom
- ✅ **Mermaid** - Interactive diagrams

## 🌐 Deploy

### GitHub Pages
```bash
mkdocs gh-deploy
```

### Build Static Site
```bash
mkdocs build
# Output in site/ directory
```

## 🆘 Common Commands

```bash
# Serve with live reload
mkdocs serve

# Build static site
mkdocs build

# Clean and rebuild
mkdocs build --clean

# Deploy to GitHub Pages
mkdocs gh-deploy

# Use different port
mkdocs serve -a localhost:8001
```
